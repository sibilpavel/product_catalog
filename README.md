# Каталог товаров (Laravel + Vue + Inertia)

## Описание

Простое full-stack приложение: каталог товаров с админ-панелью.

------------------------------------------------------------------------

## Стек

-   Laravel 13
-   Vue 3 (Composition API)
-   Inertia.js
-   PostgreSQL
-   Sanctum (авторизация по токену)
-   TailwindCSS

------------------------------------------------------------------------

## Возможности

### Публичная часть

-   Список товаров с пагинацией
-   Фильтрация по категориям
-   Страница товара

### Админка

-   Авторизация (token)
-   Создание товаров
-   Редактирование товаров
-   Удаление товаров (soft delete)

### Backend

-   REST API
-   FormRequest валидация
-   API Resources
-   Связи Eloquent (Product -\> Category)
-   Soft Deletes

### Тесты

-   Feature тесты API
-   Factories для моделей

------------------------------------------------------------------------

## Авторизация

Используется Laravel Sanctum (token-based auth)

------------------------------------------------------------------------

## Установка

composer install npm install php artisan migrate --seed npm run dev php
artisan serve

------------------------------------------------------------------------

## API

### Публичные

-   GET /api/products
-   GET /api/products/{id}
-   GET /api/categories

### Авторизация

-   POST /api/login
-   POST /api/logout

### Админка

-   POST /api/products
-   PUT /api/products/{id}
-   DELETE /api/products/{id}
